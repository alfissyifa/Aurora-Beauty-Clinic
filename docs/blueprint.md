# **App Name**: Aurora Beauty Clinic

## Core Features:

- Hero Section with Tagline: Display a hero image with the tagline 'Cantik itu Percaya Diri'.
- Service Listings: Showcase services like Facial Glow, Laser Treatment, Botox, Whitening, and Acne Treatment.
- Customer Testimonials: Implement a testimonial slider to display customer reviews.
- Photo Gallery: Present a photo gallery in a 3x3 grid layout.
- Consultation Booking Form: Enable users to book consultations by filling out a form.
- WhatsApp CTA Button: Implement a floating WhatsApp button for direct communication.
- Book Appointment Endpoint: Create an API endpoint to save booking data (name, WA number, email, service, date, notes) to Firestore.
- Fetch Services and Testimonials: Create API endpoints to retrieve service and testimonial data from Firestore.
- Email Notifications: Send automatic email notifications to the admin upon new bookings.

## Style Guidelines:

- Primary color: Soft pink (#F4B4C3) to evoke femininity and calmness.
- Background color: Very light pink (#F9E6EB) to maintain the feminine feel without being too overpowering.
- Accent color: Gold (#D4AF37) to add a touch of luxury and elegance.
- Body and headline font: 'Alegreya', a serif font, providing an elegant, intellectual, contemporary feel.
- Use elegant, minimalist icons related to beauty and skincare.
- Modern and elegant layout with a focus on whitespace and clear visual hierarchy.
- Subtle animations using framer-motion to enhance user experience, such as smooth transitions and hover effects.
